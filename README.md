# AI娱乐-孩子更像谁 🧬

基于 Google Gemini 视觉大模型的趣味亲子特征分析工具。上传一家三口的照片，AI 将深度解析五官遗传密码，通过可视化的方式为您揭示孩子更像爸爸还是妈妈。

## ✨ 核心功能

### 🔍 深度特征分析
AI 能够精准识别并对比 106 个面部关键特征点，包括：
*   **眉眼神韵**：眉形走势、眼距宽窄、双眼皮特征。
*   **鼻唇形态**：鼻梁高低、鼻翼宽窄、唇形轮廓。
*   **脸型轮廓**：下颌线条、颧骨高低、整体脸型比例。
*   **综合气质**：基于五官组合的整体神态相似度分析。

### � 炫酷可视化报告
*   **雷达扫描动画**：沉浸式的 AI 分析过程展示。
*   **特征连线图**：直观的连线图谱，将孩子面部特征与父母直接关联，一目了然。
*   **详细解读卡片**：每一个相似点都有生动的文字解读，告诉您像在哪、有多少分相似。

### 🔒 隐私安全至上
我们深知家庭照片的私密性，因此我们将隐私保护放在首位：
*   **阅后即焚**：分析结果为一次性阅后即焚。页面关闭或退出后，所有数据（包括照片和报告）将立即销毁，无法找回。
*   **一码一机**：激活码与设备绑定，防止他人冒用您的分析机会。
*   **数据自动清理**：服务器端设有严格的定时清理机制，确保没有任何用户数据残留。

## � 使用流程

1.  **获取激活码**：输入您的专属激活码开启分析之旅。
2.  **上传照片**：分别上传孩子、父亲、母亲的正面清晰生活照（支持截取和预览）。
3.  **智能分析**：点击分析，等待数秒，AI 即可完成亿级特征比对。
4.  **查看结果**：保存精美的分析报告图片到相册，与家人朋友分享这份趣味发现。

*注：本应用仅供娱乐参考，遗传学极其复杂，AI 分析基于视觉相似度，不代表生物学亲子鉴定结果。*
